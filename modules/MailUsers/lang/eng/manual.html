Mail Users
The mail users module allows the site admin to send messages to individual users or to all registered users. All fields are required fields.
Mail User Pick which users to send email to or tick the mail all users option to send a message to all regsitered users.
From Who should this email appear to originate from.
Reply-to-address What reply-to address should be set in the email.
Subject What subject should the email use.
Message This is where you actually put the text and content of the message.
