FUNCTION DESCRIPTION
Insert SmartCard DataThis function pastes chipcard-data from clipboard into your form.
Required:
Client-Chipcard-Reader and a client-program, which pushs data from chipcard into clipboard.
Note:
Using this function only "directly after" reading the card.
Structure of data must identify with form's structure.
You can see this structure in a file, which be saved with function SAVE.
Content RecoverIf you leave a page, all editor's contents will go away, too.
When you go back, you'll see a empty editor.
Don't panic, use this function to recover all data.
It works exactly like function Insert Smart Data.
